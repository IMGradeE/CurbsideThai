Bridger College

In this activity I wrote a script that constructs and inserts the body's outline (table of contents) to a pre -defined element on the page (an aside, in this case). This is done by executing a breadth-first tree traversal that adds each heading element's text content to an anchor, links the anchor to the heading, and then adds the anchor as a list item to an ordered list. These ordered lists are nested according to their heading level (higher is deeper). This makes document navigation much easier, and can really only be improved by making the table of contents' have a fixed position on and allowing users to scroll through overflow content.
